This is just a private practice and memo to use Git and Github.

(a)

  • Local side
    1. Make new topic branch on origin/master.
    2. Make some changes to source code.
    3. Commit them to local repository.
    4. Repeat form 2. some times.
    5. In topic barnch, push self to git server.
    • Server side:
      1. A compare & pull request becomes available.
      2. Do merge the pull request.
    • Return local side:
      1. Do fetch from server.
      2. Do fast forward with remote/origin/master.
      3. Complete.

(b)

  1. Do change some source code without topic branch.
  2. Commit them to local repository.
  3. Repeat form 2 some times.
  4. In topic branch, push self to git server.
  5. On git server, a compare & pull request becomes available.
  6. Do merge the pull request on git server.
